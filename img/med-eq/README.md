# メディカル事業 画像ファイル

## 必要な画像ファイル

このディレクトリには以下の画像ファイルを配置してください：

### KVブロック用画像

1. **kv_bg.jpg**
   - 用途: KVセクションの背景画像
   - 配置: 横リピート、中央配置
   - 推奨サイズ: 横幅1920px以上推奨
   - 形式: JPEG

2. **kv_inner_bg.png**
   - 用途: KV内側コンテナの背景画像
   - 配置: contain、中央配置（デスクトップ）
   - 推奨サイズ: 980px幅に最適化
   - 形式: PNG（透過対応の場合）

3. **kv_logo.png**
   - 用途: ロゴ画像
   - 配置: 左側（デスクトップ）/ 中央上部（モバイル）
   - モバイルサイズ: 画面幅の約53%
   - 形式: PNG推奨（透過背景）

4. **kv_copy.png**
   - 用途: 「IMAGING-INTELLIGENCE」コピー画像
   - 配置: コピーブロック内、中央揃え
   - 形式: PNG推奨（透過背景）

## 画像の取得元

参照サイト: https://www.nagase.co.jp/med-eq/index.html

画像は以下の方法で取得できます：
1. 参照サイトの開発者ツールで画像URLを確認
2. ブラウザで画像を右クリック → 「名前を付けて保存」
3. 元のサイトのアセットディレクトリから直接ダウンロード

## 配置後の確認

画像配置後、以下を確認してください：
- 画像が正しく表示されるか
- 背景画像が適切に配置されているか
- モバイル表示でレイアウトが崩れていないか
- 画像のアスペクト比が維持されているか

## プレースホルダー

画像が未配置の場合、EDSは自動的にalt属性のテキストを表示します。
実際の画像を配置するまで、基本的なレイアウト確認は可能です。
