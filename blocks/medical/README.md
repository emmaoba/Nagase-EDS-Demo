# Medicalブロック - 製品情報リンクセクション

## 概要

製品情報ページへのリンクボタンを実装したブロックです。
画像とテキストの2カラムレイアウトで、ホバー効果付き。

参照: https://www.nagase.co.jp/med-eq/index.html

## 📝 オーサリング方法

### index.mdでの記述

**方法1: テーブル形式（推奨）**

```markdown
## Medical

| ![](./assets/btn_dmr_img.jpg) | 医療関係者向け製品情報 |
|-------------------------------|---------------------|
| [リンク](./dmr/) | 米国やイスラエルのベンチャー技術を中心とした先進画像処理システムを取り扱っております。<br>今後も診断支援技術等に注力してまいります。 |
```

**方法2: シンプル形式**

```markdown
## Medical

[![](./assets/btn_dmr_img.jpg)](./dmr/)

医療関係者向け製品情報

米国やイスラエルのベンチャー技術を中心とした先進画像処理システムを取り扱っております。  
今後も診断支援技術等に注力してまいります。
```

### 構成要素

**1行目: 画像（リンク付き）**
```markdown
[![](./assets/btn_dmr_img.jpg)](./dmr/)
```
- `[![]()]()` = 画像をリンクで囲む
- `.img`クラスでラップされます

**2行目: タイトル**
```markdown
医療関係者向け製品情報
```
- `.ttl`クラスが付与されます
- 左に赤いボーダー表示

**3行目: 説明文**
```markdown
米国やイスラエルの...
```
- `.des`クラスが付与されます
- 改行は`<br>`または行末スペース2つ

## スタイリング仕様

### レイアウト

**デスクトップ（769px以上）:**
- Flexboxレイアウト（2カラム）
- 画像: 480px幅
- テキスト: 480px幅
- 合計: 960px（+ パディング）

**モバイル（768px以下）:**
- 縦積みレイアウト
- 画像: 100%幅
- テキスト: 100%幅、パディング5.33vw

### デザイン要素

**コンテナ:**
- 最大幅: 980px
- ボーダー: 10px solid #ccc
- 中央配置

**タイトル:**
- フォントサイズ: 24px
- フォントウェイト: 700（太字）
- 左ボーダー: 5px solid #ea5413（赤）
- 字間: 0.05em

**ホバー効果:**
- transition: opacity 0.5s
- hover時: opacity: 0.7

## カスタマイズ

### 複数のリンクボタンを追加

同じページに複数のMainブロックを配置可能：

```markdown
## Medical

(最初のリンク)

## Medical

(2番目のリンク)
```

### リンク先の変更

```markdown
[リンク](./your-page/)
```

### テキストの編集

タイトルと説明文は自由に編集可能：

```markdown
新しいタイトル

新しい説明文がここに入ります。
```

## トラブルシューティング

### 画像が表示されない
- 画像パスが正しいか確認
- `assets/btn_dmr_img.jpg`が存在するか確認

### レイアウトが崩れる
- ブレークポイントを確認
- ブラウザの開発者ツールでCSSを確認

### ホバー効果が動作しない
- `.hv-o`クラスが正しく付与されているか確認
- CSSのtransitionプロパティを確認
