# KVブロック（キービジュアル）

## 概要

長瀬産業メディカル事業サイトのキービジュアルセクションを実装したブロックです。

参照: https://www.nagase.co.jp/med-eq/index.html

## 構成要素

### HTML構造（参照サイトより）

```html
<div class="kv">
  <div class="kv__inner">
    <div class="kv__logo">
      <img src="./img/med-eq/kv_logo.png" alt="">
    </div>
    <div class="kv__copy-block">
      <p class="copy">
        <img src="./img/med-eq/kv_copy.png" alt="IMAGING-INTELLIGENCE">
      </p>
      <p class="txt">
        検査ワークフローの効率化。<br class="SP">患者さんの身体的負担の軽減。<br>
        診断の精度向上。すべては、<br class="SP">人々の健康で豊かな暮らしのため。<br>
        画像診断のさらなる「スマート」な進化に、<br>
        長瀬産業はインテリジェント・イノベーションで貢献します。
      </p>
    </div>
  </div>
</div>
```

### マークダウン形式

トップページ（index.md）に以下を追加：

```markdown
## KV

![Logo](./img/med-eq/kv_logo.png)

![IMAGING-INTELLIGENCE](./img/med-eq/kv_copy.png)

検査ワークフローの効率化。  
患者さんの身体的負担の軽減。  
診断の精度向上。すべては、  
人々の健康で豊かな暮らしのため。  
画像診断のさらなる「スマート」な進化に、  
長瀬産業はインテリジェント・イノベーションで貢献します。
```

## 必要な画像ファイル

以下の画像を `img/med-eq/` ディレクトリに配置してください：

1. **kv_bg.jpg** - 背景画像（横リピート）
2. **kv_inner_bg.png** - 内側コンテナの背景画像
3. **kv_logo.png** - ロゴ画像
4. **kv_copy.png** - 「IMAGING-INTELLIGENCE」コピー画像

## レイアウト仕様

### デスクトップ（769px以上）
- 高さ: 400px固定
- レイアウト: Flexbox（中央揃え）
- ロゴとコピーブロックを横並び
- コピーブロック左マージン: 40px
- フォントサイズ: 14px

### モバイル（768px以下）
- 高さ: auto（コンテンツに応じて可変）
- レイアウト: 縦積み
- ロゴ幅: 53.33333vw（画面幅の約53%）
- パディング: 8vw（上下）
- フォントサイズ: 3.73333vw（レスポンシブ）
- 行間: 1.8

## スタイリング詳細

### テキストスタイル
- 行間（デスクトップ）: 2.8
- 行間（モバイル）: 1.8
- テキスト配置: 中央揃え
- トップマージン: 30px（デスクトップ）/ 2.66667vw（モバイル）

### 背景
- メイン背景: 横リピート、中央配置
- インナー背景: contain、中央配置（デスクトップ）
- インナー背景: リピート、上部配置（モバイル）

## 使用方法

1. index.mdに上記のマークダウンコンテンツを追加
2. 必要な画像ファイルを配置
3. ブロックは自動的にデコレートされます

## カスタマイズ

- 高さの調整: `.kv { height: XXXpx; }`
- フォントサイズ: `.kv__copy-block { font-size: XXpx; }`
- レイアウト調整: `.kv__inner` のflexboxプロパティ

## 注意事項

- 画像パスは相対パス（`./img/med-eq/`）を使用
- モバイルレスポンシブはvw単位で実装
- BEM命名規則を使用（`kv__`プレフィックス）
- ブレークポイントは768px/769pxで統一
