## 実装内容

### JavaScript (footer.js)

**主な機能:**
- フラグメント（footer.md）を読み込み
- 2セクション構造を生成:
  1. アドレスセクション（.addr）
  2. ナビゲーションセクション
- TOPに戻るボタンの配置
- 適切なクラス付与

**処理フロー:**
```
footer.md読み込み
↓
セクション分割
↓
.addr と .box でラップ
↓
page-topボタン追加
↓
最終HTML出力
```

### CSS (footer.css)

**参照サイトのスタイルを完全再現:**

**全体:**
- 背景: #ddd（ライトグレー）
- フォントサイズ: 90%

**アドレスセクション (.addr):**
- 背景画像: bg-footer.png（repeat-x）
- パディングトップ: 60px
- 会社名: 太字、#000
- 住所: 90%サイズ、#333

**ナビゲーションセクション:**
- 背景: #a0a0a0（ダークグレー）
- テキスト: #fff（白）
- リンク: 12px（デスクトップ）、16px（モバイル）
- ボーダー: 左右に白いボーダー（デスクトップ）
- ボーダー: 下に白いボーダー（モバイル）

**TOPに戻るボタン:**
- position: fixed
- bottom: 60px
- right: 30px

**レスポンシブ:**
- デスクトップ: ナビゲーション横並び
- モバイル: ナビゲーション縦並び

## トラブルシューティング

### フッターが表示されない
- footer.mdが正しい場所にあるか確認
- サーバーを再起動

### 背景色が表示されない
- CSSが正しく読み込まれているか確認
- ブラウザのキャッシュをクリア

### リンクが動作しない
- footer.mdのURLを確認
- 外部リンクは完全なURL（https://）を使用
