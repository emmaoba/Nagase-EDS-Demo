# Footerブロック

## 概要

長瀬産業メディカル事業サイトのフッターセクション。
会社情報、ナビゲーションリンク、コピーライトを含む。

参照: https://www.nagase.co.jp/med-eq/index.html

## オーサリング方法

### footer.md（フラグメント）

**各行の役割:**

```markdown
1行目: 会社名
（空行）
2行目: 住所
（空行）
---（区切り線 - アドレスセクションとナビゲーションセクションの境界）
（空行）
3行目: ナビゲーションリンク（|で区切る）
（空行）
4行目: コピーライト
```

### 実際の記述例

```markdown
長瀬産業株式会社 NVC室　メディカル事業推進チーム

〒100-8142　東京都千代田区大手町2-6-4 常盤橋タワー

---

- [当社からの重要なお知らせ](https://www.nagase.co.jp/important/)
- [個人情報保護方針](https://www.nagase.co.jp/privacy/)
- [当サイトご利用にあたって](https://www.nagase.co.jp/attention/)
- [ソーシャルメディアポリシー](https://www.nagase.co.jp/socialmediapolicy/)
- [長瀬産業WEB](https://www.nagase.co.jp/)

Copyright © 2025 NAGASE & CO., LTD. All Rights Reserved.
```

## 構成要素

### 1. アドレスセクション（上部）
- **1行目**: 会社名 → `<h3><a>`として表示
- **2行目**: 住所 → `<p>`として表示
- 背景: 薄いグレー (#ddd) + 都市シルエット画像

### 2. ナビゲーションセクション（下部）
- **3行目以降**: ナビゲーションリンク → `<nav><ul><li>`として表示
  - オーサリング: 箇条書き形式（`-`で始まる）
  - 出力: 横並び、パイプ（|）区切り
  - 例: `| 当社からの重要なお知らせ | 個人情報保護方針 | ...`
- **最終行**: コピーライト → `<p>`として表示
- 背景: ダークグレー (#a0a0a0)

## スタイリング仕様

### カラースキーム

**アドレスセクション:**
- 背景: #ddd（薄いグレー）
- 背景画像: bg-footer.png（都市シルエット、repeat-x）
- テキスト: #333（ダークグレー）

**ナビゲーションセクション:**
- 背景: #a0a0a0（ダークグレー）
- テキスト: #fff（白）
- ボーダー: 白（1px solid）

### フォントサイズ

- 会社名: 14px（デスクトップ）、13px（モバイル）
- 住所: 13px（デスクトップ）、12px（モバイル）
- ナビゲーション: 14px（共通）
- コピーライト: 11px（デスクトップ）、10px（モバイル）

### レイアウト

**デスクトップ（769px以上）:**
- ナビゲーション: 横並び（flexbox）
- パイプ区切り: `| リンク1 | リンク2 | リンク3 |`
- フォントサイズ: 14px
- 行間: 1.8
- パイプとテキスト間のマージン: 8px

**モバイル（768px以下）:**
- ナビゲーション: 縦並び
- パイプ: 非表示
- フォントサイズ: 14px
- マージン: 4px 0

## 編集方法

### 会社名を変更
```markdown
新しい会社名
```
1行目を編集

### 住所を変更
```markdown
〒000-0000　新しい住所
```
2行目を編集

### ナビゲーションリンクを追加
```markdown
- [リンク1](URL1)
- [リンク2](URL2)
- [新しいリンク](新URL)
```
箇条書き形式で追加（`-`で開始）

### コピーライト年を更新
```markdown
Copyright © 2026 NAGASE & CO., LTD. All Rights Reserved.
```
4行目を編集

## 必要な画像

### bg-footer.png
- 用途: アドレスセクションの背景（都市シルエット）
- 配置: `img/bg-footer.png`
- リピート: 横方向（repeat-x）

## トラブルシューティング

### フッターが表示されない
1. footer.mdが正しい場所にあるか確認
2. サーバーを再起動
3. ブラウザのキャッシュをクリア

### レイアウトが崩れる
1. footer.mdの各行が正しい順序か確認
2. `---`の区切り線が正しく配置されているか確認
3. ナビゲーションリンクが箇条書き形式（`-`で開始）になっているか確認

### 背景画像が表示されない
1. `img/bg-footer.png`が存在するか確認
2. パスが正しいか確認
3. 画像ファイルが破損していないか確認
